# AI Content Studio 🚀

A SaaS-style AI content generator that creates blogs, captions, emails, and ads using Groq LLM API.

## 🔥 Features
- AI blog/caption/email generator
- Prompt suggestion shortcuts
- Word counter
- Copy to clipboard
- History of generated content
- Clean SaaS-style UI

## 🧠 Tech Stack
- HTML
- CSS
- JavaScript
- Groq API (Llama 3.1 8B Instant)

## ⚙️ How it works
User enters topic → selects tone & type → AI generates content using Groq LLM → result is displayed instantly.
